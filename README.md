# Bakage Presentation

Presentation slides for Bakage, a package manager for Prolog systems.
Presented at [The 3rd Scryer Prolog Meetup](https://hsd-pbsa.de/veranstaltung/scryer-prolog-meetup-2025/).

## Download PDF

The compiled PDF is available in [the releases section](https://github.com/constraintAutomaton/bakage-scryer-prolog-meetup-presentation/releases).

## Building

This presentation is built with [Typst](https://typst.app/).

Compile the slides:
```bash
typst compile main.typ
```

Watch for changes:
```bash
typst watch main.typ
```
